# 📧 Outlook Teams Notification Bot

Бот для автоматического мониторинга корпоративной почты Outlook (Shared Mailboxes), фильтрации тикетов ServiceNow (INC, RITM, SLA) и алертов мониторинга (SolarWinds, Zabbix) с последующей отправкой уведомлений в Microsoft Teams.

## 🚀 Основные возможности

- **Мониторинг в реальном времени**: Проверка входящих писем каждые 60 секунд.
- **Умная фильтрация**: 
  - Разделение по тегам стран: `[UZ]`, `[KZ]`, `[KG]`.
  - **Middle East Routing**: Автоматическая пересылка тикетов ОАЭ, Катара и Саудовской Аравии в отдельный канал по выделенному вебхуку.
  - Для Узбекистана ([UZ]) разрешены **только Инциденты и SLA**. Остальные типы и дубли игнорируются.
  - Автоматический пропуск системного «мусора» (Zabbix, Work Orders, SCTASK).
  - Игнорирование отмененных пользователем задач (**Withdrawn**) и уведомлений о закрытии.
- **Weekend Silence**: Бот не будет тегать (упоминать) сотрудников в выходные дни, чтобы не беспокоить их в нерабочее время.
- **Еженедельный отчет**: Автоматический сбор данных и отправка сводки по новым сотрудникам (NPR) и увольнениям (ER) каждую пятницу в **18:00 по времени Астаны**.
- **Автоматический деплой (CI/CD)**: Настроен пайплайн через GitHub Actions — любые изменения в коде сами улетают на сервер.
- **Docker-контейнеризация**: Полная изоляция и простой запуск одной командой.

## 🛠 Установка и настройка

### Переменные окружения (.env)
Создайте файл `.env` в корне проекта:
```env
CLIENT_ID=ваш_azure_client_id
CLIENT_SECRET=ваш_azure_client_secret
TENANT_ID=ваш_azure_tenant_id
TARGET_EMAIL=shared_mailbox@company.com
TEAMS_WEBHOOK_URL=https://outlook.office.com/webhook/...
TEAMS_REPORT_WEBHOOK_URL=https://outlook.office.com/webhook/... (опционально для отчетов)
```

### Запуск через Docker (рекомендуется)
```bash
docker-compose up --build -d
```

### Локальный запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите бота: `python bot.py`
3. При первом запуске перейдите по ссылке в консоли для авторизации через Microsoft Graph API.

## 🤖 CI/CD Пайплайн
Бот настроен на автоматическое обновление сервера при каждом `git push` в ветку `main`. 
Требуемые секреты в GitHub: `SSH_HOST`, `SSH_USER`, `SSH_PRIVATE_KEY`.
